# Homebrew Formula

Generation, commit and push of `poof` Formula is handled automatically by
[homebrew-tap](https://github.com/pirafrank/homebrew-tap).

## Supported platforms

The Formula supports platforms `brew` is available on:

- macOS
  - Intel (`x86_64`)
  - Apple Silicon (`aarch64`)
- Linux
  - `x86_64`
  - `aarch64`
